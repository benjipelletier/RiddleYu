# benji.codes

Monorepo for two Chinese language learning games.

## Projects

### [謎語 RiddleYu](./riddleyu)
A daily 成语 (four-character idiom) puzzle game. Solve 4 riddles, chain the characters, uncover the idiom. Inspired by Wordle.

### [歌詞了解](./gecijielong)
A Chinese lyrics learning app.

## Development

Each project is independent. Navigate into the project folder to work on it:

```bash
cd riddleyu && npm install && npm run dev
cd gecijielong && npm install && npm run dev
```
